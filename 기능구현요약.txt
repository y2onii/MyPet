1. PetRegistration → PetList 데이터 연동 기능
========================================

🔹 구현 내용:
- PetRegistration 페이지에서 등록한 반려동물 정보가 PetList 페이지에 자동으로 추가되는 기능 구현

🔹 기술적 구현:
- Context API를 사용한 전역 상태 관리
- src/contexts/PetContext.jsx 생성
- PetProvider로 앱 전체를 감싸서 상태 공유
- localStorage를 통한 데이터 영속성 보장

🔹 주요 파일:
- src/contexts/PetContext.jsx (새로 생성)
- src/App.jsx (PetProvider 추가)
- src/pages/PetRegistration.jsx (Context 사용하도록 수정)
- src/pages/PetList.jsx (등록된 데이터 표시 기능 추가)

🔹 주요 기능:
- 등록된 펫들이 PetList 맨 앞에 표시
- 등록된 동물 수 별도 표시
- 기존 더미 데이터와 함께 페이지네이션 동작
- 등록 후 자동으로 PetList 페이지로 이동

========================================
구현된 주요 Context들
========================================

1. PetContext
   - pets: 등록된 펫 목록
   - addPet: 새 펫 등록 함수
   - removePet: 펫 삭제 함수
   - totalPets: 총 펫 개수

2. AuthContext
   - user: 현재 사용자 정보
   - isLoggedIn: 로그인 상태
   - isLoading: 로딩 상태
   - login: 로그인 함수
   - logout: 로그아웃 함수

✅ 부드러운 호버 애니메이션으로 UI/UX 향상
